# A demo Go server to test Google OAuth

## Instructions to Run:

### Without live reloading:

Requirements:
- at least Go 1.17

```sh
$ go run ./cmd/server/main.go
```

### With live reloading

Requirements:
- at least Go 1.22

```sh
$ go install github.com/air-verse/air@latest # install air
$ air # run the app
```
